void MocoDataLogger::dumpMocoParamsToLogfile ( void  )

Dumps the stored data to the logfile set in setParamsLogFileName. If the file already exists it is overwritten.

Format of resulting file (3.4f):

transX transY transZ rotX rotY rotZ
transX transY transZ rotX rotY rotZ
...
