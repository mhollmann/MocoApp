- (void) clearDataAndGraphs

Clear the graph plots and remove the stored values for the transformation. Sets the counter for stored scans to zero.

- (void) updateGraphs

Update the graphs controlled by this class. This one must be called each time after values are added that should be shown. This should be called in the MAIN LOOP of the application, because this refreshs also the connected views.
